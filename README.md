# Personal Expense Tracker

## Description
This is a personal expense tracker that allows a person to add, view, and save expenses while also setting and tracking budgets. There are 5 components to this project as you'll see broken down in the Python file:
- STEP 1: ADDING AN EXPENSE
- STEP 2: VIEWING AN EXPENSE
- STEP 3: SET AND TRACK A BUDGET
- STEP 4: LOADING FROM A CSV
- STEP 5: INTERACTIVE MENU

Expenses are saved to a CSV between sessions so there's continuity.

## Features
I used loops, defined functions, created an interactive menu—to name a few of the concepts. Here's what it can do:
- Add and categorize daily expenses
- View all recorded expenses
- Set and track a monthly budget
- Save and load expenses from a CSV file

## How to Run
Use Python 3. You can run the program by opening the file in your editor of choice and running the program. It'll check to see if expenses are already available before allowing you to interact with the menu.
